Интерактивная игра в формате викторины. Есть вопрос и 4 ответа к нему.

При нажатии на ответ: 
- если ответ правильный - подсветить зелёным, 
- если неправильный, подсветить красным и подсветить зелёным правильный вариант.
Затем отобразить внизу кнопку перехода на следующий вопрос

Стек: React + Typescript + TailwindCSS

Вопросы не должны повторяться в одной игровой сессии

Загружать вопросы из files/questions.json
Формат вопросов:
[
 "question": "text",
 "options": [
   ["option": "answer"],
   ["option2": "answer2"],
   [
     "option3": "answer3",
     "right": true     
   ]
 ]
]

Оформить дизайн в стиле FAR

Для каждого вопроса брать случайный фон из директории files/backgrounds
