Интерактивная игра в формате викторины для пятничного дейлика.
Делалась исключительно по фану, ни на что не претендует и полна баянов и фактов за чью достоверность никто не ручается.

Вопросы и порядок ответов в случайном порядке.

После ответа на вопрос под вопросом отображается случайное изображение в духе "воспоминание разблокировано"

## Запуск

```shell
npm install
npm run dev
```

будет запущен веб-сервер, в консоль напишется адрес вида
```
  ➜  Local:   http://localhost:5173/
```
адрес нужно открыть в браузере

## Разворачивание

```shell
npm run build
```
